Detailed Description

Interface definition for QDaq GPIB plugins.

A GPIB plugin is created by sub-classing QDaqGpibPlugin and implementing all its virtual functions.

The interface defines a number of GPIB specific functions that are needed in order to operate a GPIB interface according to the QDaqInterface class.
